## Platform notes

- **Dev.to & Peerlist** — paste the markdown above as-is; links render automatically.
- **Medium** — Medium auto-links bare URLs, so the list works. For the bold labels, type the label text, then select it and use the link tool (Cmd/Ctrl + K) to attach the URL. Or just keep it simple: one bare URL per line with a short label in front.
- **LinkedIn article** — bare URLs auto-link; paste as plain lines. Also add your "Originally published on Dev.to →" line at the very top: https://dev.to/tanay_joshi_04/never-lose-a-training-run-again-a-checkpoint-and-resume-playbook-for-ephemeral-gpus-2m1j
- **X thread** — no need to add this block; the last tweet already carries the link. (Optionally pin the repo link in a reply.)
